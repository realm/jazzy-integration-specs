ResponseContentTransformer

public struct ResponseContentTransformer<InputContentType,OutputContentType>: ResponseTransformer

A simplified ResponseTransformer that deals only with the content of the response entity, and does not touch the surrounding metadata.

If processContent(_:) throws or returns nil, the output is an error.

If the input entity’s content does not match the InputContentType, the response is an error.

  • Processor

    A closure that both processes the content and describes the required input and output types.

    The closure can throw an error to indicate that parsing failed. If it throws a Siesta.Error, that error is passed on to the resource as is. Other failures are wrapped in a Siesta.Error.

    Parameters

    skipWhenEntityMatchesOutputType

    When true, if the input content already matches OutputContentType, the transformer does nothing. When false, the tranformer always attempts to parse its input. Default is true.
    transformErrors

    When true, apply the transformation to Error.content (if present). When false, only parse success responses. Default is false.
