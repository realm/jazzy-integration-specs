ResponseContentTransformer

public struct ResponseContentTransformer<InputContentType, OutputContentType> : ResponseTransformer

A simplified ResponseTransformer that deals only with the content of the response entity, and does not touch the surrounding metadata.

If processEntity(_:) throws or returns nil, the output is an error.

If the input entity’s content does not match the InputContentType, the response is an error.

  • Processor

    A closure that both processes the content and describes the required input and output types.

    The input will be an Entity whose content is safely cast to the type expected by the closure. If the response content is not castable to InputContentType, then the pipeline skips the closure and replaces the resopnse with a RequestError describing the type mismatch.

    The closure can throw an error to indicate that parsing failed. If it throws a RequestError, that error is passed on to the resource as is. Other failures are wrapped in a RequestError.
